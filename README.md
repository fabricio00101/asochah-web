# ASOCHAH Web

> Proyecto web de práctica personal desarrollado durante el aprendizaje de HTML y CSS.

## Descripción
Este proyecto es una práctica para crear un sitio web institucional ficticio llamado ASOCHAH. Contiene secciones como institucional, formulario de contacto, imágenes y términos y condiciones, con el objetivo de practicar la organización de contenido y estilos.

## Tecnologías utilizadas
- HTML
- CSS
- Imágenes locales

## Objetivo de aprendizaje
- Practicar la creación de un sitio web estructurado por secciones.
- Aplicar estilos CSS a diferentes elementos y páginas.
- Mejorar la organización de archivos y recursos.
- Desarrollar un diseño simple pero funcional.

## Contenido del proyecto
- `index.html` → Página principal.
- `style.css` → Estilos del sitio.
- Carpetas como `imagenes/`, `formulario/`, `institucional/` y `terminosycondiciones/`.
